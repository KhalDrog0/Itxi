Tipos de Degradados (Transiciones de color)

DEGRADADO LINEAL

Dirección fija: cielo → montaña

Uso: Fondos de diapositiva, barras de progreso

DEGRADADO RADIAL

Desde el centro: grano de café

Uso: Énfasis en elementos centrales, fondos dramáticos

📍 Procedimiento Paso a Paso: Aplicar Degradado
  1. Seleccionar la forma haciendo clic sobre ella
  2. Ir a: Formato de forma → Relleno de forma → Degradado
  3. Elegir tipo: Lineal (direccional) o Radial (circular)
  4. Definir puntos de degradado: Mínimo 2 (color de inicio y color de fin)
  5. Seleccionar colores: Usar gotero para capturar colores de imágenes reales
  6. Ajustar ángulo: Para degradados lineales (0°=izq→der, 90°=arriba→abajo)
  7. Ajustar transparencia si es necesario (especialmente para fondos)
Ejercicio práctico: Crear un fondo que evoque el amanecer en una finca cafetera del Cauca usando degradado lineal de azul cielo (#87CEEB) a verde montaña (#228B22).

☕ Práctica Guiada: Del Grano a la Taza

Crearemos una presentación completa de 5 diapositivas sobre el proceso del café, aplicando todas las herramientas aprendidas. Esta será tu entrega final del período.

Estructura de la Presentación

N° Diapositiva Contenido específico Elementos a usar
1 Portada Título creativo, nombre del estudiante, imagen representativa del café Degradado radial, texto con sombra, forma de grano
2 Introducción ¿Qué es el café? Origen histórico en el Cauca, importancia cultural SmartArt de jerarquía, mapa mental, imágenes
3 Proceso inicial Etapas: Siembra, germinación, crecimiento, floración Formas de flujo, flechas conectoras, iconos

Guía Paso a Paso: Diapositiva 1 (Portada)

📍 Instrucciones detalladas:
  1. Diseño de fondo: Insertar rectángulo que ocupe toda la diapositiva. Aplicar degradado radial: centro claro (hueso #F5F5DC) a bordes oscuros (café #3E2723)
  2. Título: Insertar cuadro de texto → "Del Grano a la Taza: El Café del Cauca" → Fuente: Arial Black, 40pt, color café oscuro, sombra suave
  3. Subtítulo: "Proceso de transformación del café" → Fuente: Arial, 24pt, color verde bosque, cursiva
  4. Elemento decorativo: Crear forma de grano de café: Insertar óvalo estirado + pequeña forma de luna invertida para la mella. Agrupar (Ctrl+G)
  5. Posicionar: Título centrado en tercio superior, subtítulo en tercio medio, grano en esquina inferior derecha con rotación de 15°
  6. Efecto final: Aplicar sombra exterior suave al grano para profundidad

Guía Rápida: Diapositivas 2-5

Diap 2 - Introducción:
Insertar → SmartArt → Ciclo básico. Añadir: Origen Etiopía → Llegada a Colombia → Región del Cauca. Cambiar colores a tonos café/verde.
Diap 3 - Proceso:
Insertar → Formas → Formas de flujo. Secuencia: Óvalo (Inicio) → Rectángulo (Proceso) → Rombo (Decisión) → Flechas de conexión.
Diap 4 - Transformación:
Insertar → Gráfico → Columnas. Datos: Etapas vs Tiempo (horas). Personalizar colores a degradado café.
Diap 5 - Cierre:
Insertar imagen de campesinos → Formato → Enviar atrás. Rectángulo negro encima → Transparencia 40%. Texto blanco centrado.

📍 Procedimiento: Guardar como PDF (Obligatorio para entrega)
  1. Archivo → Guardar como → Examinar (o Guardar como Adobe PDF si está instalado)
  2. En "Tipo", seleccionar PDF (*.pdf) del menú desplegable
  3. Configurar opciones:
    • Publicar: Diapositivas (no Notas ni Handouts)
    • Rango: Todo
    • Calidad: Estándar (para pantalla) o Alta (para impresión)
  4. Clic en "Opciones" para verificar que "Diapositivas ocultas" NO esté marcado
  5. Clic en "Publicar" o "Guardar"
  6. Verificar que el archivo PDF se abre correctamente y se ven todas las diapositivas
📍 Procedimiento: Guardar como Video (Opcional avanzado)
  1. Archivo → Exportar → Crear un video
  2. Calidad de video: Seleccionar HD (720p) (suficiente para proyección escolar)
  3. Grabar narraciones: No (a menos que hayan grabado audio)
  4. Segundos por diapositiva: Establecer 5 segundos (ajustar según cantidad de texto)
  5. Clic en "Crear video"
  6. Elegir ubicación y nombre del archivo
  7. Esperar a que termine la barra de progreso (puede tardar varios minutos)
Entrega final requerida: Dos archivos: (1) Presentación editable .pptx y (2) Versión PDF para visualización. Nombrar: "ApellidoNombre_Grado10_Cafe.pptx"
