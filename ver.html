🌿 La Época de Gusanos (Seip)

En la cosmovisión Nasa, la época de Gusanos representa el tiempo de la transformación interior, del trabajo profundo en la tierra, de preparar el terreno para lo que vendrá.

¿Qué significa Seip?

  • Transformación: Como el gusano que se convierte en mariposa
  • Trabajo profundo: Labor en lo invisible, en la raíz
  • Paciencia: Procesos que no se apresuran
  • Preparación: Construir bases para el futuro
┌─────────────────────────────────────┐ │ CICLO DE SEIP │ │ │ │ 🌱 → 🐛 → 🌀 → 🦋 → 🌸 │ │ Siembra Trabajo Capullo Vuelo Floración │ profundo interno libre compartida │ │ │ "El gusano no tiene prisa, │ │ pero tampoco descansa" │ └─────────────────────────────────────┘
El café también tiene sus épocas: siembra, floración, desarrollo del fruto, maduración. Cada etapa requiere su tiempo y cuidado, como nuestra presentación digital.
📍 Metodología: Círculo de Palabra

Actividad: En círculo, cada estudiante comparte una palabra que asocie con "transformación" o "trabajo profundo". El docente conecta con el proceso de crear presentaciones.

📍 Procedimiento: Aplicar Degradado
  1. Seleccionar la forma
  2. Ir a: Formato de forma → Relleno de forma → Degradado
  3. Elegir tipo: Lineal o Radial
  4. Definir puntos de degradado: Mínimo 2 (inicio y fin)
  5. Seleccionar colores: Usar gotero para capturar colores de imágenes
  6. Ajustar ángulo: Para degradados lineales
